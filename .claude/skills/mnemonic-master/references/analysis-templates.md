# Mnemonic Analysis & Creation Templates

## Single Mnemonic Analysis Template

```markdown
## Analysis: Question [ID]

**Question**: [full question text]
**Correct Answer**: [answer]
**Current Mnemonic**: "[existing mnemonic]"

### CRISP Score (1-5 each):

| Criterion | Score | Analysis |
|-----------|-------|----------|
| **C**oncise | X/5 | [Is it under 10 words? Easy to say quickly?] |
| **R**elevant | X/5 | [Does it directly lead to the answer?] |
| **I**mmediate | X/5 | [Can recall answer in <2 seconds?] |
| **S**ound-based | X/5 | [Uses phonetics, rhyme, alliteration?] |
| **P**ictureable | X/5 | [Creates a clear mental image?] |

**Total Score**: X/25

### Technique Used:
[Identify: Acronym / Rhyme / Phonetic / Visual / Analogy / Story / etc.]

### Verdict:
[ ] Excellent (21-25) - Keep as-is
[ ] Good (16-20) - Minor tweaks optional
[ ] Needs Work (11-15) - Should improve
[ ] Replace (0-10) - Fundamentally flawed

### Issues Identified:
- [Issue 1]
- [Issue 2]

### Improved Version:
**New Mnemonic**: "[improved version]"

**Techniques Applied**: [list techniques used]

**Why Better**: [explanation]
```

## Quality Checklist (Pre-Deployment)

Before finalizing any mnemonic:

```markdown
## Final Check: Question [ID]

**Mnemonic**: "[text]"

### Functionality Tests:
- [ ] 2-Second Test: Can recall answer within 2 seconds?
- [ ] Stress Test: Works under exam anxiety?
- [ ] Uniqueness Test: Won't confuse with similar concepts?
- [ ] Self-Contained: Works without additional explanation?

### Audience Tests:
- [ ] Appropriate for adult learners?
- [ ] No offensive content?
- [ ] Culturally neutral?
- [ ] No insider jargon?

### Technical Tests:
- [ ] Factually accurate?
- [ ] Leads to CORRECT answer (not trap answer)?
- [ ] Doesn't oversimplify dangerously?

### Format Tests:
- [ ] Under 15 words?
- [ ] No run-on sentences?
- [ ] Clear punctuation?
- [ ] Consistent style with others in category?

**APPROVED**: [ ] Yes [ ] No - needs revision
**Reviewer**: [name/date]
```
